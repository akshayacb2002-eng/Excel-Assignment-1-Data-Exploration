Excel Assignment 1 using SUM, COUNT, IF, SUMIF, COUNTIF and Text Functions

# Excel Assignment 1 – Data Exploration

## Functions Used

### SUM
Calculated total price of all products = $10,100

### COUNT
Counted 34 products in dataset.

### AVERAGE

### MIN and MAX

Highest price = $1000

### IF Function
Classified products into:
- High Price
- Standard Price

### SUMIF
Total Electronics category price = $8,050

### COUNTIF
Products under $100 = 11
